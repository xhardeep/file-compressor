## Usage

1. Open `index.html` in your browser
2. Select output format (JPEG, PNG, or PDF)
3. Set target file size in KB
4. Drag & drop or upload files
5. Click Convert
6. Download compressed files

## Tech Stack

Vanilla HTML, CSS, JavaScript
PDF.js (PDF rendering)
pdf-lib (PDF manipulation)
JSZip (batch download)
